# お金と仕事の街 — つながりタウン

小学2〜4年生が、架空の街で仕事・給料・支出・貯金・税金・公共予算・お金の循環を体験する教育Webアプリです。ログイン、サーバー、APIキーは不要です。

## 主な機能

- SVGで描画した2.5D街マップと12施設の仕事・お金メモ
- 6職種（パン屋、バス運転手、看護師、ごみ収集、農家、図書館）の操作ミッション
- 子ども向け給料明細と、簡略化した税金の説明
- 必要な物、欲しい物、貯金、助け合い、街の店への支出選択
- 予想外の出来事と、公共サービスを含む複数の対応方法
- 10コインを6分野に配る街の予算会議
- タップで詳細が分かるお金の循環マップ
- 点数をつけない一週間の振り返り、仕事図鑑、保護者向けページ
- `localStorage`による自動保存、続きからの再開、進捗リセット
- キーボード操作、読み上げ、`prefers-reduced-motion`対応

アプリ内の価格、給料、税金は、金融教育のために簡略化した架空の値です。現実の価格や税率を示すものではありません。

## 起動方法

必要環境: Node.js 20.19以上（または22.12以上）

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開きます。

## ビルドと確認

```bash
npm run typecheck
npm run build
npm run preview
```

本番用ファイルは `dist/` に生成されます。Viteの `base` は相対パス (`./`) に設定しているため、GitHub Pagesのサブディレクトリ配信に対応します。

## GitHub Pagesで公開

1. リポジトリへpushします。
2. GitHub Actionsで `npm ci` と `npm run build` を実行します。
3. 生成された `dist/` をPages artifactとしてデプロイします。

手元で `gh-pages` ブランチへ配信する場合も、公開対象は `dist/` です。独自のルーターを使わない単一ページ構成のため、リロード時の404対策は不要です。

## 構成

```text
src/
  App.tsx       画面と1週間の導線
  data.ts       仕事・困りごと・支出・出来事・予算・循環の教材データ
  useGame.ts    状態遷移とlocalStorage保存
  MapScene.tsx  SVG街マップと施設データ
  Mission.tsx   6職種の操作ミッション
  Icon.tsx      統一線画アイコン
  styles.css    デザインとレスポンシブ、アクセシビリティ
  types.ts      教材・ゲーム状態の型
```

個人情報や入力内容を外部へ送る処理はありません。保存データは利用中のブラウザ内にだけ保持されます。
