# つながりタウン — はたらく・かせぐ・くらす

小学2〜4年生が、架空の街で「仕事をする → 給料をもらう → 税金を払う → 使う・貯める → 街が変わる」を体験する、3日間の教育シミュレーションゲームです。

選択肢を押すだけではなく、ドラッグ、長押し、タイミング操作で仕事そのものを遊びます。まちがえても直して最後まで働け、基本給は減りません。ログイン、サーバー、APIキーは不要です。

## 遊んでみる

- 公開版: [https://esma-dev-studio.github.io/tsunagari-town/](https://esma-dev-studio.github.io/tsunagari-town/)

## ゲームの流れ

1. 街で「食べもの」「交通」「きれいさ」の困りごとを見る
2. 仕事を選び、自分の手でミッションをクリアする
3. 給料日のミニゲームで、給料コインから税金を必要な枚数だけ街の箱へ分け、残りをまとめて財布へ入れる
4. 毎日、限られたお金を「買う・貯める・財布に残す」のどれにするか決めてから家へ帰る
5. 予想外のできごとへの対応を選び、財布、貯金、元気、時間、街の信頼がどう変わったか数字で確かめる
6. 仕事とお金の使い方によって、次の日の元気、仕事レベル、街の店、信頼が変わる
7. 3日目の終わりに、3日間で払った税金コインを「食べもの」「交通」「きれいさ」へすべて配り、次の3日間の街を動かす

おさいふと貯金は次の3日間にも引き継がれます。全部は買えない価格にしているため、「今使うか、あとに残すか」を自然に考えられます。買い物をしない日も、「貯める」「財布に残す」という自分の決定として記録されます。

## 3つの仕事ゲーム

| 仕事 | 直接操作する内容 | 街への効果 |
| --- | --- | --- |
| パン屋 | 注文どおりの材料をボウルへドラッグし、生地をぐるぐるこね、焼き色メーターを見てミトンを長押しする | 食べものを増やす |
| バス運転手 | 道路を左右にドラッグしてコーンをよけ、3つの停留所の手前からブレーキを長押しして安全に止まる | 交通を良くする |
| 資源回収 | 街のごみを3種類の箱へドラッグして分別し、回収車のレバーを長押しして適切な強さでまとめる | 街をきれいにする |

仕事の結果は星とスコアで伝えますが、失敗で行き止まりにはしません。やり直すと成功範囲が広がるなど、小さな子どもでも完了できる支援があります。仕事レベルが上がると操作にも補助が入ります。

## お金と街のシミュレーション

- 給料は「基本給 + うでまえボーナス 0〜2コイン + 街の困りごとボーナス」で計算
- 給料日は明細を読むだけではなく、税金分のコインを1枚ずつ街の箱へ分け、手取りを一度に財布へ運ぶ直接操作ゲーム
- 税金コインはドラッグでき、タップやキーボードでも同じ操作を完了できる
- 税金は学校、道路、公園などをみんなで支えるお金として扱い、3日分を「食べもの」「交通」「きれいさ」の3分野へ配る
- 買う・貯める・財布に残すのどれかを毎日決定するまで帰宅へ進まないため、お金を使わないことも選択として体験できる
- 夕ごはんの材料は翌日の元気、仕事ノートは仕事経験、街の店での買い物は食べもの、助け合いは街の信頼に反映
- 貯金額は自分で決められ、予想外のできごとへの備えとして使える
- 予想外のできごとは、選ぶ前に影響を比べ、選んだ後に財布、貯金、元気、時間、街の信頼の「前 → 後」を表示
- 仕事による街の変化、1日の収支、3日間の記録を画面で確認できる
- 仕事図鑑、街の施設マップ、保護者向けの振り返りも収録

アプリ内の価格、給料、税金は、金融教育のために簡略化した架空の値です。現実の価格、給料、税率を示すものではありません。

## 操作とアクセシビリティ

マウスとタッチ操作のほか、キーボードだけでも遊べます。

- パン屋: 材料は `Enter`、こねる操作は `Space` / `Enter` / 左右矢印、オーブンは `Space` / `Enter` の長押し
- バス: 左右矢印でハンドル、`Space` の長押しでブレーキ
- 資源回収: ごみを選んでから箱を押す操作、レバーは `Space` / `Enter` の長押し
- 給料日: 税金コインと手取りの山はドラッグ、タップ、`Enter` / `Space` で操作
- 画面名の読み上げボタンと、状態変化を伝えるライブ領域
- タッチしやすい大きなボタン、スマートフォン・タブレット・PC向けレスポンシブ表示
- `prefers-reduced-motion` によるアニメーション軽減
- `localStorage` による自動保存、続きからの再開、ホーム画面からの進捗リセット

## ローカルで起動

必要環境: Node.js 20.19以上（または22.12以上）

```bash
npm install
npm run dev
```

ターミナルに表示されたローカルURLをブラウザで開きます。

## 型チェック・ビルド

```bash
npm run typecheck
npm run build
npm run preview
```

本番用ファイルは `dist/` に生成されます。Viteの `base` は相対パス (`./`) のため、GitHub Pagesのサブディレクトリ配信に対応しています。

## GitHub Pagesへの公開

`main` ブランチへpushすると、`.github/workflows/deploy-pages.yml` が次の処理を自動実行します。

1. Node.js 22で `npm ci`
2. `npm run build`
3. `dist/` をGitHub Pages artifactとしてアップロード
4. GitHub Pagesへデプロイ

## 主な構成

```text
src/
  App.tsx                画面遷移と3日間の導線
  useGame.ts             ゲーム状態、お金の台帳、localStorage保存
  WorkplaceCenter.tsx    3つの仕事選択と操作説明
  Mission.tsx            仕事ゲーム画面への入口
  ArcadeShift.tsx        カウントダウン、仕事結果、給料計算
  arcade/
    BakeryGame.tsx       パン屋の直接操作ゲーム
    BusGame.tsx          バスのリアルタイム運転ゲーム
    WasteGame.tsx        分別と回収車の直接操作ゲーム
  TownSimulator.tsx      時間、元気、街の状態、1日・3日レポート、税金コインの配分
  PaydaySimulator.tsx    税金と手取りを分ける給料日の直接操作ゲーム
  SpendSimulator.tsx     毎日の買う・貯める・残す決定シミュレーション
  MapScene.tsx           SVGの街マップと施設データ
  data.ts                仕事、支出、できごとなどの教材データ
```

個人情報や入力内容を外部へ送る処理はありません。保存データは利用中のブラウザ内だけに保持されます。
